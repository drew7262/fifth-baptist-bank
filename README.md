Customer Portal Features

The customer-facing side provides a complete and believable online banking experience.

Account Dashboard

A central hub where customers can view all their accounts (Checking, Savings, Credit Card, Mortgage) at a glance, including balances and key details such as credit limits.

Fund Transfers

Customers can transfer money between their accounts. The system correctly handles credit card payments and prevents transfers from credit lines.

Detailed Account View

Each account includes a detailed view with:

Full transaction history

Account numbers

Interest rates and account-specific details

PDF Statements

Customers can generate and download professional-looking PDF statements directly from the account details view.

Card Services

Instantly lock or unlock debit cards

Order replacement cards

Online Credit Application

Eligible customers can apply for a new credit card using a multi-step application process.
The system simulates an automated review and queues applications for manual admin approval.

Real-Time Notifications

An in-app notification system alerts users to:

Account approvals

Security alerts

Large or pending transactions

Live Chat Support

A persistent live chat widget allows customers to communicate directly with an administrator.

Administrator Panel Features

The admin panel acts as a full command center for managing users and the banking ecosystem.

Centralized Customer Management

Admins can view, create, edit, and delete customer profiles and all associated accounts from a central interface.

Transaction Manipulation

Manual Transactions:
Admins can manually add debit or credit transactions to any account.

Transaction History Generation:
Generate years of realistic, randomized transaction history to make accounts appear aged and legitimate.

Security & Access Control

Lock or unlock user accounts with a specific reason

Enable or disable specific features per user (e.g., credit applications)

Account Review & Approval

Dedicated approval queue for new accounts (e.g., credit cards)

Admins can approve or deny applications

Approvals automatically notify customers with account details

Audible alerts notify admins of new pending reviews

Administrator Management

Supports multiple administrators. A lead admin can create, edit, or delete other admin accounts.

Internal Account Notes

Admins can add private, timestamped notes to customer profiles.
These notes are visible only to other administrators.

Live Chat Monitoring

Admins can respond to customer chats and receive audible alerts for new incoming messages.

User Event Logging

A detailed event log tracks actions such as:

Logins

Application submissions

Card actions

Global Site Banner

Admins can manage a site-wide banner for announcements such as scheduled maintenance.
